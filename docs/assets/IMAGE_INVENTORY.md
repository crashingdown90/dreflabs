### To Replace with Real Photos

1. **Profile Photo**:
   ```bash

   cp your-photo.jpg public/images/profile.jpg
   ```

2. **Project Screenshots**:
   ```bash
   # Take actual screenshots of your projects
   # Resize to 1200x630px for consistency
   cp screenshot.jpg public/projects/project-name.jpg
   ```

3. **Portfolio Screenshots**:
   ```bash
   # Capture real application screenshots
   # Recommended size: 1200x800px
   cp app-screenshot.jpg public/portfolio/category/filename.jpg
   ```

### Image Optimization Tips

- **Use WebP format** for better compression (Next.js auto-converts)
- **Optimize with tools**: TinyPNG, ImageOptim, or Squoosh
- **Recommended dimensions**:
  - Profile: 400x400px
  - Blog/Projects: 1200x630px (OG standard)
  - Portfolio: 1200x800px
  - Thumbnails: 600x400px
